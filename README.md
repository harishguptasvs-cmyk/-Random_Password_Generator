# Random Password Generator

## 📌 Description
The Random Password Generator is a Python-based application that generates secure and customizable passwords. It features a graphical user interface (GUI) that allows users to select password length and character types, along with a copy-to-clipboard functionality for convenience.

## 🚀 Features
- Generate strong and random passwords  
- Customize password length  
- Include letters, numbers, and symbols  
- GUI-based application using Tkinter  
- Copy password to clipboard feature  
- Simple and user-friendly interface  

## 🛠️ Technologies Used
- Python  
- Tkinter (GUI)  
- Built-in libraries (random, string)  

## 🎯 Purpose
This project aims to improve password security by enabling users to generate strong and unpredictable passwords with ease and convenience.

## 📂 Project Structure
- Main Python Script  
- GUI Components  
- Password Generation Logic
